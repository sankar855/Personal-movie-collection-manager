# personal-movie-collection-manager
The Personal Movie Collection Manager is a desktop application developed to organize and manage a personal movie collection efficiently. The application provides a simple graphical interface through which movie records can be stored, viewed, updated, searched, and deleted. Information such as movie title, genre, release year, language, and rating is maintained in a database for easy access and management.

The project was developed to demonstrate the implementation of CRUD (Create, Read, Update, Delete) operations and database connectivity in a real-world application. It also provides practical experience in GUI development, SQL query execution, and data management.

Key Features
-> Movie records can be added and stored in a database.
-> Stored movie details can be viewed through a graphical interface.
-> Specific movies can be searched using relevant information.
-> Existing movie records can be updated whenever required.
-> Unwanted movie records can be removed from the database.
-> Personal ratings and movie details can be maintained efficiently.

Technologies Used
Python – Application development
Tkinter – Graphical User Interface (GUI)
SQLite/MySQL – Database management
SQL – Database operations and queries
Learning Outcomes
Understanding of CRUD operations
Practical knowledge of database connectivity
Experience in SQL query execution
Basic GUI development skills
Understanding of data storage and retrieval techniques

Conclusion
The Personal Movie Collection Manager serves as a simple and effective solution for managing movie records digitally. The project demonstrates the integration of a graphical user interface with a database while providing hands-on experience in software development and database management concepts.
